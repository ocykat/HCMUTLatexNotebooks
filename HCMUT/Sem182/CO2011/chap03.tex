\chapter{Integer Program}

\hi{Integer Program}
  \par \textbox {
    \tb{An integer program}
      \begin{align*}
        \text{Maximize } & c^T x \\
        \text{subject to } & Ax \leq b \\
        & x \in Z^n
      \end{align*}

  }

\hi{LP Relaxation}
  \par An \tb{LP relaxation} of an integer program is a linear program obtain from an integer program by elminating the integer constraint.
