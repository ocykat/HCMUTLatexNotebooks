\chapter{Threads}

\hi{Overview}
  \par A \tb{thread} is a basic unit of CPU utilization; it comprises:
  \begin{itemize}
    \item \tb{A thread ID}
    \item \tb{A program counter}
    \item \tb{A register set}
    \item \tb{A stack}
  \end{itemize}
  \par  It shares with other threads belonging to the same process its code section, data section, and other operating-system resources, such as open files and signals.
